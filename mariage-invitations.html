                👥 Import — Nouvelle liste d'invités
                Utilisez cet import pour ajouter de nouveaux invités depuis votre fichier Excel initial.

                Colonnes obligatoires :
                Nom   Catégorie   NbPersonnes   InvitéCivil   InvitéReligieux

                InvitéCivil : oui ou non
                InvitéReligieux : non, vh ou repas
                📄

                Cliquez pour sélectionner le fichier CSV

                Excel → Enregistrer sous → CSV UTF-8

                Aperçu (5 premières lignes)
                📤 Télécharger le modèle
                🔄 Import — Mise à jour des réponses
                Utilisez cet import pour mettre à jour les réponses et invitations remises d'invités déjà présents dans l'application.

                Colonnes d'identification :
                Nom — obligatoire
                Catégorie — recommandé si plusieurs invités ont le même nom

                Colonnes optionnelles (laissez vide = non modifié) :

                CIVIL
                CivilReponse : oui (présent) / non (absent)
                → coche automatiquement "a répondu"

                RELIGIEUX
                ReligInvitationRemise : oui / non
                ReligReponseVH : oui / non
                ReligReponseRepas : oui / non
                → coche automatiquement "a répondu"

                MenusAdulte : nombre   MenusEnfant : nombre

                ↺ Annule et remplace Une nouvelle intégration écrase les données précédentes pour les invités concernés.
                🔄

                Cliquez pour sélectionner le fichier CSV de mise à jour

                Excel → Enregistrer sous → CSV UTF-8
